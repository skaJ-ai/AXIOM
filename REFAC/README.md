# REFAC

AXIOM 프로젝트에 대한 딥리서치 기반 분석과 리팩토링/개선 제언을 구조화한 폴더입니다.

## 구성
- `00_overview.md` — 총평 및 핵심 판단
- `01_product_analysis.md` — 디자인, 기능, 차별점, 제품성 분석
- `02_spec_vs_implementation_gap.md` — 설계도와 구현 간극 분석
- `03_openai_best_practice_mapping.md` — OpenAI 공식 문서 기반 베스트 프랙티스 매핑
- `04_priority_recommendations.md` — 우선순위 중심 개선 제언

## 목적
이 폴더의 목적은 단순 코드 리뷰가 아니라,
**제품 정의 → 설계 의도 → 구현 현실 → 개선 우선순위**를 한 번에 정리하는 것입니다.
